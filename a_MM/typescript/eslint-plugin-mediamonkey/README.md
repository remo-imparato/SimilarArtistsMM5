# eslint-plugin-mediamonkey

Custom rules for MediaMonkey

## Installation

You'll first need to install [ESLint](https://eslint.org/):

```sh
npm i eslint --save-dev
```

Next, install `eslint-plugin-mediamonkey`:

```sh
npm install eslint-plugin-mediamonkey --save-dev
```

## Usage

Add `mediamonkey` to the plugins section of your `.eslintrc` configuration file. You can omit the `eslint-plugin-` prefix:

```json
{
    "plugins": [
        "mediamonkey"
    ]
}
```


Then configure the rules you want to use under the rules section.

```json
{
    "rules": {
        "mediamonkey/rule-name": 2
    }
}
```
